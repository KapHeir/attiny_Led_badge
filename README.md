# ATtiny LED Badge

A decorative LED badge project based on the ATtiny85 microcontroller.  
This repository contains a star-shaped PCB version and a future heart-shaped PCB version.

The current completed version is the **star-shaped badge**, which uses 5 LEDs, a push button, a CR2032 battery holder, and an ATtiny85 to generate multiple LED animation patterns.

---

# Project Overview

This project was created as a decorative and educational PCB badge design using KiCad and an ATtiny85 microcontroller.

The badge is designed to:
- blink and animate multiple LEDs
- switch between animation modes using a push button
- run from a small battery
- demonstrate both PCB design and embedded control with a microcontroller

This repository is also intended as a portfolio-style project that shows:
- schematic design
- PCB layout
- Gerber generation
- GitHub project organization
- basic embedded programming with ATtiny85

---

# Features

- ATtiny85-based LED control
- 5 LED outputs
- Push button input
- On/off switch
- CR2032 battery powered
- Compact decorative PCB concept
- Designed in KiCad
- Example code included
- Expandable to multiple visual themes
